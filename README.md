# Emotion Categories

감정 단어를 분류하고 표시하는 React 컴포넌트입니다.

## 기능

- 감정 범주별 단어 분류
- 감정 강도에 따른 정렬
- 시각적 표현 (색상 코드)
- 반응형 디자인

## 사용 방법

1. 프로젝트에 컴포넌트를 추가합니다:
```javascript
import EmotionWordsList from './emotion_categories';
```

2. 컴포넌트를 렌더링합니다:
```javascript
<EmotionWordsList />
```

## 의존성

- React
- Papa Parse (CSV 파싱)
- Tailwind CSS (스타일링)
